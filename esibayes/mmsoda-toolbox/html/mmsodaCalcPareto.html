mmsodaCalcPareto
Syntax

Description

paretoScores = mmsodaCalcPareto(objScores)

This function calculates Pareto scores for a given list of N x M objective scores, N being the number of parameter samples that have been rated according to M objective functions.

The function supports 3 methods of calculating the Pareto scores:

  1. Goldberg (1989)
  2. Zitzler and Thiele (1999)
  3. Vrugt (2003)
Which method is used is determined by an optional string argument like so:

paretoScores = mmsodaCalcPareto(objScores,'g')

paretoScores = mmsodaCalcPareto(objScores,'z')

paretoScores = mmsodaCalcPareto(objScores,'v')

for Goldberg (1989), Zitzler and Thiele (1999), or Vrugt (2003) Pareto ranking, respectively. If no string argument is provided, paretoScores = mmsodaCalcPareto(objScores,'v') is assumed.
