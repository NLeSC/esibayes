Toolbox contents
mmsodaOpenPdf
Syntax

mmsodaOpenPdf()

Description

This function allows the user to open a PDF document in an external viewer from within the MATLAB help browser. Note that this does require that the command line to the PDF viewer program on the user's system need to be set in helper-apps.txt, otherwise, you will get this error.
