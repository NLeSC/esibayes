Description

mmsodaVisualization4()

Standard visualization routine that creates 3 figures using mmsodaSubplotScreen(). The upper left figure shows a mmsodaPlotGelmanRubin() result, the lower left figure shows a mmsodaPlotSeq() result, while the figure on the right shows the paramete correlation using mmsodaMatrixOfScatter() with typeStr set to 'par-par'.
