Tools
mmsodaCopyMakefileCopies the default Makefile to the current directory.
mmsodaDiffStructFunction that analyzes two configuration structs and checks if they amount to the same type of optimization. Useful for resuming an aborted optimization.
mmsodaLoadSettingsReads default values to configuration variables which the user hasn't specified but which are needed.
mmsodaOpenBrowserOpens an external web browser.
mmsodaOpenPdfOpens a PDF document in an external PDF viewer.
mmsodaParsePairsParses optional input parameter to various functions.
mmsodaPrepParallelFilesCalls mmsodaCopyMakefile and mmsodaWriteJobscript.
mmsodarootReturns the root of the MMSODA Toolbox for MATLAB.
mmsodaSubplotScreenProvides functionality similar to MATLAB's built in subplot, but acting on figures instead of on axes within a figure.
mmsodaUnpackUnpacks information from the model and objective function m-files, and assigns it to the right variables.
mmsodaVerifyFieldNamesVerifies whether the configuration variables are all valid and consistent.
mmsodaWriteJobscriptHelps you to construct PBS jobscripts and shell scripts.
subaxesProvides similar functionality as subplot, except with higher information content; less whitespace in between axes.
uimatlabReturns true if the code is executed in MATLAB.
uioctaveReturns true if the code is executed in Octave.
