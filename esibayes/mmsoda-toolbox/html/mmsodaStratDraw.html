mmsodaStratDraw
Syntax 
sDraw = mmsodaStratDraw(conf)
Description

mmsodaStratDraw divides each axis (1..conf.nOptPars) into X segments, such that X is the largest positive integer for which X^conf.nOptPars < conf.nSamples. This way the parameter space is divided into nStratSamples hyperrectangles. Exactly one sample is taken at the center of each hyperrectangle. When the number of required samples conf.nSamples is larger than nStratSamples, an additional nSamples-nStratSamples samples are taken using uniform random sampling of the parameter space (see mmsodaUnifRandDraw).
