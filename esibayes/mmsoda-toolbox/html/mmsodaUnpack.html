Description

mmsodaUnpack()

This is usually the second line of the main model file (see here) as well as the second line of the objective function (see here). mmsodaUnpack() uses information from the configuration to assign the values from init, parVec, and constants to the corresponding variables as they are used inside the model and objective function.
