mmsodaEvolveComplex
Syntax 
[curCompl,acceptedChild] = mmsodaEvolveComplex(conf,curCompl,propChild)
Description
This function evolves a complex through a Metropolis annealing scheme, that is by accepting or rejecting a proposed child based on (1) the objective score of that child and (2) a uniform random draw. Output variable curCompl contains the updated complex. Depending on how the annealing works out, acceptedChild contains either the proposed child propChild, or the member of curSeq that has the maximum evaluation number (column conf.evalCol). acceptedChild is an output argument because it needs to be added to the sequences variable later on.
