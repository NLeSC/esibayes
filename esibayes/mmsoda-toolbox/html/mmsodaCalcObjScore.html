mmsodaCalcObjScore
Syntax 
[evalResults,nModelEvals] = mmsodaCalcObjScore(conf,evalResults,nModelEvals)
Description
Function mmsodaCalcObjScore iterates over the parameter sets in variable evalResults; if a parameter combination in evalResults has not yet been evaluated, function mmsodaCalcObjScore first calls the model using the model call string specified by conf.modelName. The model result is directed towards an objective function as specified by conf.objCallStr (single-objective) or conf.objCallStrs (multi-objective). Both the model function itself and the objective function(s) are called using an eval statement. Individual objective functions should be constructed in such a way that they yield a 1x1 array objective score (see here). For single-objective optimization, this value needs to be a log likelihood for the Metropolis scheme to be sensibly applied.
