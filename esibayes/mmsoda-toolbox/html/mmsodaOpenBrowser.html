Toolbox contents
mmsodaOpenBrowser
Syntax

mmsodaOpenBrowser()

Description

This function allows the user to open a website in an external browser from within the MATLAB help browser. Note that this does require that the command line to the browser program on the user's system need to be set in helper-apps.txt, otherwise, you will get this error.
