Error: pdfviewer not set

Apparently, you clicked on some link that normally opens a PDF document. However, it seems that you haven't set the command line with which the PDF viewer is started. On Windows it is usually something like:

"C:\Program Files (x86)\Adobe\Reader 10.0\Reader\AcroRd32.exe" %s

for opening the PDF in Adobe's Acrobat Reader. Note that the %s is because the URL to the PDF document is filled in using an sprintf() call. If you open the helper-apps.txt file and add the command line to your PDF viewer after the pdfviewer= bit, you should be able to open PDFs directly from the documentation.
