mmsodaGenerateOffspring
Syntax 
propChild = mmsodaGenerateOffspring(conf,curCompl,curSeq)
Description
Based on the covariance matrix of the current complex (local variable covCompl), a jumping distance jumpDist is calculated as the product of:
  1. a random draw from a normal distribution of dimension conf.nOptPars;
  2. the jump distance scaling factor conf.jumpRate;
  3. the covariance matrix of the current complex covCompl.
The jump is centered on local variable jumpBase, which can be one of the following:
  1. the mean of the current complex;
  2. the last point (i.e. the point in the current sequence that has the highest iteration number).
