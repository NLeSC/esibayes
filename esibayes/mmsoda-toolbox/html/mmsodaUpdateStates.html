Description

stateValuesKFPost = mmsodaUpdateStates(conf,stateValuesKFPert,stateValuesKFPost,stateValuesKFInn,iDAStep)

Updates the state values by adding the innovations to the perturbed prior state values, thus creating the posterior state values.
