mmsodaVerifyFieldNames
Syntax

mmsodaVerifyFieldNames(conf,'check')

mmsodaVerifyFieldNames(conf,'show')

authorizedFieldNames = mmsodaVerifyFieldNames(conf,'return')

Description

mmsodaVerifyFieldNames(conf,'check')

The mmsodaVerifyFieldNames() function verifies whether the configuration is consistent and whether the configuration variables are all valid ones.

mmsodaVerifyFieldNames(conf,'show')

Prints a list of the authorized options but checks nothing.

authorizedOptions = mmsodaVerifyFieldNames(conf,'return')

Returns a cell array of string containing the valid option names.
