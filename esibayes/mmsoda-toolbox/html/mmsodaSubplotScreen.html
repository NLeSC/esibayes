mmsodaSubplotScreen
Syntax

mmsodaSubplotScreen(A,B,C)

mmsodaSubplotScreen()

mmsodaSubplotScreen(A,B,C,'figureNumber',D)

Description

mmsodaSubplotScreen(A,B,C)

does for figures what MATLAB's built-in subplot does for axes. For example:

mmsodaSubplotScreen(1,2,1)

creates a empty figure window on the left side of the screen. Similarly 

for k=1:6
   mmsodaSubplotScreen(3,2,k)
end

creates 6 empty figure windows, arranged in 3 rows and 2 columns on the screen.

mmsodaSubplotScreen()

If no input arguments are supplied, mmsodaSubplotScreen(1,1,1) is assumed.

mmsodaSubplotScreen(A,B,C,'figureNumber',D)

You can use the 'figureNumber'option to identify a certain figure. For example: 

for k=1:6
   mmsodaSubplotScreen(3,2,k,'figureNumber',1000+k)
end

creates figures 1001:1:1006.
