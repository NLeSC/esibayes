mmsodaDiffStruct
Syntax

structdiff = mmsodaDiffStruct(struct1,struct2,ignoreFields)

Description

This function is used when resuming from an earlier MMSODA run. It checks whether certain key configuration variables such as modeStr or nCompl have not accidentally been changed between runs.
