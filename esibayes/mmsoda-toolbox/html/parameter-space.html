Parameter space

Consider a model which has one parameter, such as for example in the bimodalmodel example. Valid parameters for this model come from the real number space, which in this case is one-dimensional since the model only has 1 parameter. Because of practical considerations such as number precision, our parameter cannot take just any value, but rather it has to be constrained to a certain range within the real number space:

With θ the parameter set, which in the case of our example contains only one parameter; Θ which represents all the numbers from the valid range of the number line; R, which represents all numbers from a one-dimensional real number space.
