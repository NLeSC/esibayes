sodaGelmanRubin
Syntax 
rHatRoot = sodaGelmanRubin(conf,sequences)
Description

The sodaGelmanRubin function estimates the scale reduction that could be achieved if additional samples were being drawn indefinitely. The procedure is based on variances that may exist between different parameters in the same sequence, as well as corresponding parameters in different sequences. See also Gelman and Rubin (1992) Inference from iterative simulation using multiple sequences Statistical Science 7(4):457-472.

The structure array conf must have a field modeGelRub, containing one of two strings: 'strict' or 'loose'. When 'strict' is used, the equation used to calculate rHatRoot is exactly the same as that used in the beforementioned paper; if 'loose' is selected, a slightly modified version is used. The difference between the two relates to the denominator of the middle term of Equation (4) as included on page 461 of the paper; in 'strict' mode the denominator is set to m*n^2, whereas in 'loose' mode, (m*n)^2 is used.
