partcomplexes
Syntax 
complexes = partcomplexes(scemPar,evalResults)
Description 
complexes = partcomplexes(scemPar,evalResults)
Function partcomplexes partitions the rows from variable evalResults over the scemPar.nCompl Markov Chains (aka complexes or sequences). This is achieved in a two-step process:
  1. ordering the rows from evalResults by improving objective score (as recorded in the evalResults(:,scemPar.objCol) column);
  2. subsequent assignment of every ith entry from this ordered array to the ith complex.
