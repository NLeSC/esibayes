Toolbox contents
sodaContinue
Syntax 
runCondition = sodaContinue(conf,nModelEvals)
Description
This function determines whether the optimization procedure should continue. Termination is based on one of the following:
  1. the maximum number of iterations as specified by conf.nModelEvalsMax has been reached;
  2. the optimization end time as specified by conf.optEndTime has been exceeded;
  3. convergence has been achieved.
