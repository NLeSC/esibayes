The objective function
Syntax
function objScore = calcLikelihood(conf,constants,allStateValuesKF,allValuesNOKF,parVec)
Description
function objScore = calcLikelihood(conf,constants,allStateValuesKF,allValuesNOKF,parVec)

When constructing the objective function, make sure to strictly adhere to the format as displayed above; only the name of the function itself (here: calcLikelihood) may be different. The reason behind this strictness is that the optimization algorithm prepares these variables and then constructs the call to the objective function at runtime. Also, the sodaUnpack() function call, that is typically found on the second line of the objective function, expects the variables conf, constants and parVec to be present in the objective function workspace.

Argument Description 
objScore

Scalar double output argument, containing the log-likelihood score for single-objective optimization, or just any type of objective score for multi-objective optimization.

 
conf

1x1 struct argument, in which your MMSODA configuration variables have been collected, using the information from the './results/conf.mat' file as well as the default settings from the file indicated by conf.useIniFile.

 
constants

1x1 struct argument, in which the model constants have been collected, using the information from './data/constants.mat'.

 
allStateValuesKF

Input argument of size 1 x conf.nMembers x conf.nStatesKF x conf.nPrior and of class double, containing the values of all Ensemble Kalman Filter members for all Ensemble Kalman Filter states for all time steps listed in conf.priorTimes. 

allValuesNOKF

Input argument of size 1 x conf.nMembers x conf.nNOKF x conf.nPrior and of class double, containing the values of all Ensemble Kalman Filter members for all Non-Ensemble Kalman Filter variables for all time steps listed in conf.priorTimes. 

parVec

Input argument of size 1 x conf.nOptPars and class double, containing the parameter combination that MMSODA sampled from the parameter space.
