prepseqarray
Syntax 
sequences = prepseqarray(scemPar,sequences)
Description
Function prepseqarray prepares the array sequences for the next generation by pre-allocation. This is achieved by concatenating the existing array sequences with an array of size [scemPar.nOffspringPerCompl,size(sequences,2),scemPar.nCompl], filled with NaNs.
