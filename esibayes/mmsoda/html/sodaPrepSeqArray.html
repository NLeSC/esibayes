sodaPrepSeqArray
Syntax 
sequences = sodaPrepSeqArray(conf,sequences)
Description
Function sodaPrepSeqArray prepares the array sequences for the next generation by pre-allocation. This is achieved by concatenating the existing array sequences with an array of size [conf.nOffspringPerCompl,size(sequences,2),conf.nCompl], filled with NaNs.
