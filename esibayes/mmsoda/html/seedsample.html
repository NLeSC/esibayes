seedsample
Syntax 
thisMachineSeed = seedsample(seedFileName)
Description

thisMachineSeed = seedsample(seedFileName)

Initializes the pseudorandom number generator by reading from the file seedFileName. Each line in this file must be formatted according to:
<computername>,<randomseed><linebreak>
<computername>,<randomseed><linebreak>
...

In case reading fails for some reason, a warning is issued and the random seed is initialized using the seconds field returned by clock.

Note that the computername may be retrieved from the underlying operating system's environmental variables list by: 

>>getenv('COMPUTERNAME')
