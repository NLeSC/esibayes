sodaStratRandDraw
Syntax 
sRandDraw = sodaStratRandDraw(conf)
Description

Stratified random sampling of points from the valid parameter space as defined by conf.parSpaceLoBound and conf.parSpaceHiBound.

sodaStratRandDraw divides each axis (1..conf.nOptPars) into X segments, such that X is the largest positive integer for which X^conf.nOptPars < conf.nSamples. This way the parameter space is divided into nStratRandSamples hyperrectangles. From each hyperrectangle, exactly one random sample is taken. When the number of required samples conf.nSamples is larger than nStratRandSamples, an additional nSamples-nStratRandSamples samples are taken using uniform random sampling of the parameter space (see sodaUnifRandDraw).
