Description

sodaMargHist(conf,evalResults)

This function displays histograms of each optimized parameter.

sodaMargHist(conf,evalResults,...)

Various aspects of the function can be tweaked by including parameter value pairs (see table below).

Parameter Description
'histMode' String argument 'absolute' or 'relative' defines whether the vertical axis of the histogram shows absolute numbers or relative numbers, respectively. The y-axis label will show count when 'histMode' is 'absolute' or frequency when 'histMode' is 'relative'.
'nBins' Integer scalar defines how many bins should be used in constructing the histogram.
'iterSelection' Vector of integers specifying which model evaluations should be used in constructing the histogram.
'faceColor' Color value of the bin faces. Example: sodaMargHist(conf,evalResults,'faceColor',[0,0,1]) for blue colored bin faces.
'edgeColor' Color value of the bin edges. Example: sodaMargHist(conf,evalResults,'edgeColor',[0,0.5,0]) for olive colored bin edges.
'fixedBinEdges' Logical scalar indicating whether the bin edges should be fixed to the lower and upper limits of the parameter space, or whether the bin edges should zoom to the minimum and maximum values in the histogram.
'showNumberOfPoints' Logical scalar. If set to true, the number of points that were used in constructing the histogram is displayed in the plot's title.
