Toolbox contents
plotconv
Syntax 
plotconv(scemPar,evalResults)
Description

For each calibration parameter, plotconv makes a subplot, in which it visualizes the iteration number on the x-axis versus the sampled parameter value on the y-axis. Near x=1, the sampling extent should span the full extent of the constrained parameter space, but as the optimization progresses, this band should become smaller, unless a parameter is not identifiable from the available data.
