Description 
plotgelmanrubin(scemPar,critGelRub)

For each parameter, this function plots how the Gelman-Rubin convergence criterion decreases with each generation. 

plotgelmanrubin(scemPar,critGelRub,'view','full')
plotgelmanrubin(scemPar,critGelRub,'view','zoom')

Including the view option lets you set the limits on the y-axis. If you select 'full', no limits are set and all data points will be visible. If you select 'zoom', the y-axis automatically zooms to the range spanned by the last scemPar.nGelRub generations in the critGelRub array. Default behavior is 'zoom'.
