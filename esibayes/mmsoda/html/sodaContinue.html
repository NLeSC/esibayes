continuescem
Syntax 
runCondition = continuescem(scemPar,nModelEvals)
Description
This function determines whether the optimization procedure should continue. Termination is based on one of the following:
  1. the maximum number of iterations as specified by scemPar.nModelEvalsMax has been reached;
  2. the optimization end time as specified by scemPar.optEndTime has been exceeded;
  3. convergence has been achieved.
