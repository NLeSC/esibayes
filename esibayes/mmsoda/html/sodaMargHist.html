Description 
sodaMargHist(conf,evalResults)
This function displays histograms of each optimized parameter. The 'ylims' of each subplot are automatically set to the validity range as specified by scemPar.rangeMin and scemPar.rangeMax.

sodaMargHist(conf,evalResults,'absolute')
sodaMargHist(conf,evalResults,'relative')
Inclusion of the additional parameter generates plots with absolute counts or frequencies, respectively. Default is 'absolute'.
