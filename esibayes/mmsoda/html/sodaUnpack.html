sodaUnpack()

This is usually the second line of the main model file (see here) as well as the second line of the objective function (see here). sodaUnpack() uses information from the configuration to assign the values from stateValuesKFOld, valuesNOKFOld, parVec, and constants to the corresponding variables as they are used inside the model and objective function.
