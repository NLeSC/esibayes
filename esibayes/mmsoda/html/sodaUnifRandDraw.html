Description

uDraw = sodaUnifRandDraw(conf,'parSpace')

Uniform random sampling of points from the valid parameter space as defined by conf.parSpaceLoBound and conf.parSpaceHiBound.

uDraw = sodaUnifRandDraw(conf,'stateSpace')

Uniform random sampling of points from the valid state space as defined by conf.stateSpaceLoBound and conf.stateSpaceHiBound.
