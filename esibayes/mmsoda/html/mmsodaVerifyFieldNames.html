sodaVerifyFieldNames
Syntax

sodaVerifyFieldNames(conf,'check')

sodaVerifyFieldNames(conf,'show')

authorizedFieldNames = sodaVerifyFieldNames(conf,'return')

Description

sodaVerifyFieldNames(conf,'check')

The sodaVerifyFieldNames() function verifies whether the configuration is consistent and whether the configuration variables are all valid ones.

sodaVerifyFieldNames(conf,'show')

Prints a list of the authorized options but checks nothing.

authorizedOptions = sodaVerifyFieldNames(conf,'return')

Returns a cell array of string containing the valid option names.
