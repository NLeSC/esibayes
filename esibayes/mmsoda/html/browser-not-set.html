Error: browser not set

Apparently, you clicked on some link that normally opens a website in your browser. However, it seems that you haven't set the command line with which the browser is started. On Windows it is usually something like:

"C:\Program Files (x86)\Mozilla Firefox\Firefox.exe" %s

"C:\Program Files (x86)\Internet Explorer\iexplore.exe" %s

for opening the website in Mozilla Firefox and Microsoft Internet Explorer, respectively. Note that the %s is because the URL to the website is filled in using an sprintf() call. If you open the helper-apps.txt file and add the command line to your browser after the browser= bit, you should be able to open external websites directly from the documentation.
