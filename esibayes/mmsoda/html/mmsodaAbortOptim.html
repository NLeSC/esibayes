Syntax

Description

converged = sodaAbortOptim(critGelRub,conf)

This function returns true if there are at least conf.nGelRub values in the critGelRub variable, which must be sufficiently close (argument conf.convMaxDiff) and less than conf.critGelRubConvd. It returns false otherwise.
