sodaPrepRejArray
Syntax 
metropolisRejects = sodaPrepRejArray(conf,metropolisRejects)
Description

metropolisRejects = sodaPrepRejArray(conf,metropolisRejects)

Function sodaPrepRejArray prepares the array metropolisRejects for the next generation by pre-allocation. This is achieved by concatenating the existing array metropolisRejects with an array of size [conf.nOffspringPerCompl,size(sequences,2),conf.nCompl], filled with NaNs.
