sodaCalcObjScore
Syntax 
[evalResults,nModelEvals] = sodaCalcObjScore(conf,evalResults,nModelEvals)
Description
Function sodaCalcObjScore iterates over the parameter sets in variable evalResults; if a parameter combination in evalResults has not yet been evaluated, function sodaCalcObjScore first calls the model using the model call string specified by conf.modelName. The model result is directed towards an objective function as specified by conf.objCallStr (single-objective) or conf.objCallStrs (multi-objective). Both the model function itself and the objective function(s) are called using an eval statement. Individual objective functions should be constructed in such a way that they yield a 1x1 array objective score. For single-objective optimization, this value needs to be a log likelihood for the Metropolis scheme to be sensibly applied.
