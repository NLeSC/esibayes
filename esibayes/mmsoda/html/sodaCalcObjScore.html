calcobjscore
Syntax 
[evalResults,nModelEvals] = calcobjscore(scemPar,evalResults,nModelEvals)
Description
Function calcobjscore iterates over the parameter sets in variable evalResults; if a row k in evalResults has not yet been evaluated (evalResults(k,scemPar.evalCol)==false), function calcobjscore first calls the model using the model call string specified by scemPar.modelCallStr. The model result is directed towards an objective function as specified by scemPar.objCallStr. Both the model function itself and the objective function are called using an eval statement. The objective function should be constructed in such a way that it yields a 1x1 array objective score.
