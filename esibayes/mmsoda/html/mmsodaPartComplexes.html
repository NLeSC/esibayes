sodaPartComplexes
Syntax 
complexes = sodaPartComplexes(conf,evalResults)
Description 
complexes = sodaPartComplexes(conf,evalResults)
Function sodaPartComplexes partitions the rows from variable evalResults over the conf.nCompl Markov Chains (aka complexes or sequences). This is achieved in a two-step process:
  1. ordering the rows from evalResults by improving objective score (as recorded in the evalResults(:,conf.objCol) column);
  2. subsequent assignment of every ith entry from this ordered array to the ith complex.
